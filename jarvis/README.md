# JARVIS
An AI named as jarvis that can perform many tasks.

# How to use it

1. create a virtual environment ~
 ``python -m venv .venv``
 
2. activate the virtual environment ~
 ``.venv\Scripts\activate``
 
3. install the requirements ~
 ``pip install -r requirements.txt``
 
4. run the jarvis ( /jarvis/ ) dir ~
 ``python -m jarvis.main``


# Wake Word

In the `config.py` file, you can change the wake word to your preference.

the default wake word is "hello", in voice option
